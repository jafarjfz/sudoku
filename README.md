# sudoku
simple practice project in C. basic sudoku game with blank board for the player to fill.

the project was written to practice function usage, nested loop usage, two dimensional array usage, data validation and console display. it's pretty straightforward, the project generates a blank 9x9 canvas divided into 3x3 sections. the game allows the player to choose the slot (horizontally and vertically) and proceeds to check whether said slot is available by classic sudoku rules. the board is displayed after each move and the player wins once the entire board is filled according to the rules of the game.

in order to quit the game and close the console, the player can input a negative number.
